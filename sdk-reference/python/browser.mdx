# Dendrite Browser

The `DendriteBrowser` class manages a browser instance using Playwright, allowing interactions with web pages using natural language. It handles initialization with API keys, manages browser contexts, and provides methods for navigation, authentication, and other browser-related tasks.

## Installation

```bash
pip install dendrite-sdk
```

## Usage

```python
from dendrite_sdk import DendriteBrowser

async with DendriteBrowser() as browser:
    page = await browser.goto("https://example.com")
        # Perform actions on the page
        ```

        ## Constructor

        ```python
        DendriteBrowser(
            id=None,
                openai_api_key: Optional[str] = None,
                    dendrite_api_key: Optional[str] = None,
                        anthropic_api_key: Optional[str] = None,
                            playwright_options: Any = {
                                    "headless": False,
                                            "args": STEALTH_ARGS,
                                                }
                                                )
                                                ```

                                                Creates a new instance of the DendriteBrowser.

                                                ### Parameters

                                                - `id` (Optional[str]): An optional ID for the browser instance. If not provided, a UUID is generated.
                                                - `openai_api_key` (Optional[str]): The OpenAI API key. If not provided, it's fetched from the environment variables.
                                                - `dendrite_api_key` (Optional[str]): The Dendrite API key. If not provided, it's fetched from the environment variables.
                                                - `anthropic_api_key` (Optional[str]): The Anthropic API key. If not provided, it's fetched from the environment variables.
                                                - `playwright_options` (Any): Options for configuring Playwright. Defaults to running in non-headless mode with stealth arguments.

                                                ### Raises

                                                - `Exception`: If any of the required API keys (Dendrite, OpenAI, Anthropic) are not provided or found in the environment variables.

                                                ## Methods

                                                ### get_active_page

                                                ```python
                                                async def get_active_page() -> DendritePage[DownloadType]
                                                ```

                                                Retrieves the currently active page managed by the PageManager.

                                                #### Returns

                                                - `DendritePage`: The active page object.

                                                #### Raises

                                                - `Exception`: If there is an issue retrieving the active page.

                                                ### goto

                                                ```python
                                                async def goto(
                                                    url: str,
                                                        new_page: bool = False,
                                                            timeout: Optional[float] = 15000,
                                                                expected_page: str = ""
                                                                ) -> DendritePage[DownloadType]
                                                                ```

                                                                Navigates to the specified URL, optionally in a new tab.

                                                                #### Parameters

                                                                - `url` (str): The URL to navigate to.
                                                                - `new_page` (bool): Whether to open the URL in a new page. Defaults to False.
                                                                - `timeout` (Optional[float]): The maximum time (in milliseconds) to wait for the page to load. Defaults to 15000.
                                                                - `expected_page` (str): A description of the expected page type for verification. Defaults to an empty string.

                                                                #### Returns

                                                                - `DendritePage`: The page object after navigation.

                                                                #### Raises

                                                                - `Exception`: If there is an error during navigation or if the expected page type is not found.

                                                                ### authenticate

                                                                ```python
                                                                async def authenticate(domains: Union[str, list[str]])
                                                                ```

                                                                Authenticates the browser for the specified domains.

                                                                #### Parameters

                                                                - `domains` (Union[str, list[str]]): A domain or list of domains to authenticate.

                                                                #### Raises

                                                                - `Exception`: If authentication fails.

                                                                ### new_page

                                                                ```python
                                                                async def new_page() -> DendritePage[DownloadType]
                                                                ```

                                                                Opens a new page in the browser.

                                                                #### Returns

                                                                - `DendritePage`: The newly opened page.

                                                                #### Raises

                                                                - `Exception`: If there is an issue opening a new page.

                                                                ### add_cookies

                                                                ```python
                                                                async def add_cookies(cookies)
                                                                ```

                                                                Adds cookies to the current browser context.

                                                                #### Parameters

                                                                - `cookies` (List[Dict[str, Any]]): A list of cookies to be added to the browser context.

                                                                #### Raises

                                                                - `Exception`: If the browser context is not initialized.

                                                                ### close

                                                                ```python
                                                                async def close()
                                                                ```

                                                                Closes the browser and uploads authentication session data if available.

                                                                #### Raises

                                                                - `Exception`: If there is an issue closing the browser or uploading session data.
