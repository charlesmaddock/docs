## Installation

Install the Dendrite SDK. We also need to run `dendrite install` to install the browser drivers.

With pip:

```
pip install dendrite-sdk && dendrite install
```

With poetry:

```
poetry add dendrite-sdk && poetry run dendrite install
```

## Hello world

For our first Dendrite script, let's start the Dendrite client, go to google.com and enter "hello world" into the search field:

```python
from dendrite_sdk import Dendrite

# Get your Dendrite API key from dendrite.systems
DENDRITE_API_KEY = "..."

def hello_world():
  # Initate the Dendrite client
  client = Dendrite(dendrite_api_key=DENDRITE_API_KEY)

  # Navigate with the `goto` method
  client.goto("https://google.com")

  # Populate the search field
  client.fill("Search input field","hello world")

hello_world()
```

<Note>
  **Pro tip:** If you have a variable in you .env file named DENDRITE_API_KEY,
  you don't have to provide it when initiating the Dendrite class.
</Note>
